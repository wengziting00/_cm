本程式的目標是撰寫一個可通用的函數，用來求解任意階之常係數齊次線性常微分方程。根據微分方程理論，此類方程的解完全由其特徵方程的根所決定，因此程式首先利用 numpy.roots 計算由輸入係數所形成的特徵多項式之所有根。由於數值計算可能產生極小的誤差（例如理論上的實根在計算後出現極小的虛部），程式會先對根進行誤差修正，將虛部接近零的根視為實根，以確保分類的正確性。接著，利用 Counter 統計各特徵根的重複次數，因為在常係數齊次線性微分方程中，根是否為重根以及重根的次數，會直接影響通解中所需乘上的多項式因子。對於實根，程式依其重根次數產生
