Geometry Toolkit (Python)

本專案實作了一組 二維幾何計算工具庫，包含點、直線、圓等物件的表示方式，以及常見幾何問題的演算法，例如：兩直線交點、直線與圓交點、兩圓交點、垂足計算、畢氏定理驗證等。內容適合作為幾何計算、電腦圖學、競賽幾何等用途的基礎工具。

📌 功能總覽
幾何物件

Point(x, y)：點

Line(a, b, c)：一般式直線 ax + by + c = 0

Circle(center, r)：圓

Triangle(A, B, C)：三角形

幾何運算

兩直線交點：line_intersection(L1, L2)

直線與圓交點：line_circle_intersection(line, circle)

兩圓交點：circle_intersection(c1, c2)

一點到直線垂足：foot_of_perpendicular(line, p)

畢氏定理驗證：verify_pythagoras(A, B, C)

幾何變換

平移：translate_point(p, dx, dy)

縮放：scale_point(p, sx, sy)

旋轉：rotate_point(p, theta)（以原點為中心）

🧩 解題流程與設計理念

本工具庫透過由簡到繁的方式構築幾何計算框架：

1. 定義基本幾何物件

程式以 Point、Line、Circle 作為基底，使後續計算能以物件的方式操作，而非純數值推算。
例如：Line.from_points(p1, p2) 可自動求出直線一般式。

2. 兩直線交點

利用二元一次方程組解法（克拉瑪公式）求交點。
若判別式 D = a1*b2 - a2*b1 為 0 ⇒ 平行或重合無交點。

3. 直線與圓交點

將直線代入圓方程，化為一元二次方程：

判別式 < 0：無交點

判別式 = 0：相切

判別式 > 0：兩交點

採用代數法 + 幾何方式簡化（平移圓心）。

5. 垂足（投影）

對點 P 投影到直線 ax + by + c = 0。
公式來自向量投影與直線法向量 (a, b) 的性質。

6. 畢氏定理驗證

利用距離平方（避免開根號誤差）驗證：

AB² = AC² + BC²


若成立則 C 為直角頂點。

7. 幾何變換

實作平移、縮放、旋轉，讓幾何物件能參與變換運算。
旋轉採用標準旋轉矩陣：

[x']   [ cosθ  -sinθ ] [x]
[y'] = [ sinθ   cosθ ] [y]
